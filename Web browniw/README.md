# Web Browniw

Simplified local PHP web project (brownie theme) with login, registration, user management.
Place the `Web browniw` folder in your server root (htdocs) and open http://localhost/Web%20browniw/

Registration fields and storage:
fname|mname|lname|age|mobile|email|birthdate|gender|username|password|course|role

Courses: CS, IT, THM
Role auto-assigned: CS/IT -> COMSOC, THM -> HM
